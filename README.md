# Dara

Autonomous security agent and coding assistant with Python CLI.

## Features
- Rich CLI with memory management (vector DB), project status, suggestions
- Canary integrity checks for critical MD files (soul, journal, parking lot, etc.)
- Heartbeat for continuity
- Secure, self-contained with canary-protected persistence

## Usage
```bash
python3 DARA_CLI.py
python3 DARA_CLI.py project status
python3 DARA_CLI.py memory add "note here"
python3 DARA_CLI.py canary
```

See `DARA_CLI.py` for all commands.

Repo for easy fixes and contributions.

Part of the Dara ecosystem (see also [dara-cli](https://github.com/jimmotes2024/dara-cli)).
